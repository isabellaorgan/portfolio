PROJECTS

CADET

Learning management system designed for use by Code Fellows instructors. Cadet features integration with GitHub, and provides the ability to create a course repo with complete curricum in seconds. Cadet utilises a custom built npm markdown parser, Cadet Parser, which reads a string from a markdown file and generates a course object from the file. Cadet also features a custom built command line interface, cadet-cli, which uses Cadet Parser to duplicate a group of repos from the command line. Collaborative project completed in approximately 60 hours using a Scrum workflow and Trello, JSON and GitHub OAuth. Primary lead on UI design comp using Sass.

COLO(U)RBLINDR

Responsive web application that allows a colorblind user to filter a photo based on their profile color blindness type to show color sighted people how they see the world. Collaborative project completed in approximately 60 hours, using Agile workflow and Trello, Node/Express.js, MongoDB and TDD. Primary contributor to UI design (HTML, CSS, Responsiveness), also worked with Node/Express, and MongoDB.

HOUR LONG LUNCH

Responsive web application showing the location of neighborhood food trucks on an interactive map, with a focus on showing the user travel and wait times for the purpose of getting back to work within an allotted time. A collaborative project completed in four days using an Agile workflow and Trello, Google Maps, jQuery and JSON parsing. Project originator and manager. Contributed to UI design (HTML, CSS, Responsiveness), Google Maps, JavaScript.
